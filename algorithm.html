Add Event
Time: O(n)Space: O(1)▼
Algorithm
  1. Prompt user for event name and category.
  2. Allocate a new node using malloc().
  3. Fill in name, category, auto-generate date and time.
  4. Assign incremented ID to the node.
  5. If list is empty, set head = newNode.
  6. Else traverse to the last node (temp→next == NULL).
  7. Set lastNode→next = newNode.
  8. Push newNode onto the undo stack.
  9. Increment event_count.
Pseudo Code
function addEvent():
    newNode ← allocate Event
    newNode.name ← input()
    newNode.category ← input()
    newNode.date ← getCurrentDate()
    newNode.time ← getCurrentTime()
    newNode.id ← next_id++
    newNode.next ← NULL

    if head == NULL:
        head ← newNode
    else:
        temp ← head
        while temp.next ≠ NULL:
            temp ← temp.next
        temp.next ← newNode

    pushToStack(newNode)
    event_count++

Delete Event
Time: O(n)Space: O(1)▼
Algorithm
  1. Prompt user for event ID.
  2. If head is NULL, print "empty list" and return.
  3. If head→id matches, set head = head→next, free old head.
  4. Else traverse with prev and current pointers.
  5. When current→id matches, set prev→next = current→next.
  6. Free the matched node.
  7. Decrement event_count.
Pseudo Code
function deleteEvent(id):
    if head == NULL: return "Empty"

    if head.id == id:
        temp ← head
        head ← head.next
        free(temp)
        event_count--
        return

    prev ← head
    curr ← head.next
    while curr ≠ NULL:
        if curr.id == id:
            prev.next ← curr.next
            free(curr)
            event_count--
            return
        prev ← curr
        curr ← curr.next

    print "Not found"

Edit Event
Time: O(n)Space: O(1)▼
Algorithm
  1. Prompt user for event ID.
  2. Traverse linked list searching for matching ID.
  3. If found, prompt for new name and category.
  4. Overwrite the existing fields.
  5. If not found, display error.
Pseudo Code
function editEvent(id):
    temp ← head
    while temp ≠ NULL:
        if temp.id == id:
            temp.name ← input("New name")
            temp.category ← input("New category")
            return
        temp ← temp.next
    print "Not found"

Search Event
Time: O(n)Space: O(1)▼
Algorithm
  1. Prompt user for search keyword.
  2. Convert keyword to lowercase.
  3. Traverse the list; for each node, convert name to lowercase.
  4. Use substring matching (strstr) for partial matches.
  5. Display all matching events.
  6. If none found, print "No results".
Pseudo Code
function searchEvent(keyword):
    found ← 0
    temp ← head
    key ← toLower(keyword)
    while temp ≠ NULL:
        name ← toLower(temp.name)
        if name contains key:
            display(temp)
            found++
        temp ← temp.next
    if found == 0:
        print "No results"

Display Timeline
Time: O(n)Space: O(1)▼
Algorithm
  1. Check if head is NULL; if yes, print "No events".
  2. Print table header row.
  3. Set temp = head.
  4. While temp ≠ NULL, print event details.
  5. Advance temp = temp→next.
Pseudo Code
function displayTimeline():
    if head == NULL:
        print "No events"
        return
    print "ID | Name | Category | Date | Time"
    temp ← head
    while temp ≠ NULL:
        print temp.id, temp.name, temp.category,
              temp.date, temp.time
        temp ← temp.next

Today's Timeline
Time: O(n)Space: O(1)▼
Algorithm
  1. Get today's date string.
  2. Traverse the linked list.
  3. Compare each node's date with today.
  4. If match, display the event.
  5. Count matches; if zero, show "No events today".
Pseudo Code
function displayTodayTimeline():
    today ← getCurrentDate()
    count ← 0
    temp ← head
    while temp ≠ NULL:
        if temp.date == today:
            display(temp)
            count++
        temp ← temp.next
    if count == 0:
        print "No events today"

Replay Day
Time: O(n)Space: O(n)▼
Algorithm
  1. Create a temporary stack.
  2. Traverse entire linked list, pushing each event onto the temp stack.
  3. Pop every element from the temp stack and display it.
  4. This shows events in reverse order (LIFO).
  5. Free the temporary stack.
Pseudo Code
function replayDay():
    tempStack ← empty
    temp ← head
    while temp ≠ NULL:
        push(tempStack, temp)
        temp ← temp.next

    print "=== Replay (Reverse Order) ==="
    while tempStack not empty:
        event ← pop(tempStack)
        display(event)
    freeStack(tempStack)

Undo Event
Time: O(n)Space: O(1)▼
Algorithm
  1. Check if undo stack is empty; if yes, print "Nothing to undo".
  2. Pop the top event from the stack.
  3. Extract its ID.
  4. Search the linked list for that ID and delete the node.
  5. Free the stack node and the popped event copy.
Pseudo Code
function undoLastEvent():
    if stack_top == NULL:
        print "Nothing to undo"
        return
    event ← popFromStack()
    deleteEventById(event.id)
    free(event)
    print "Event undone"

Statistics
Time: O(n)Space: O(1)▼
Algorithm
  1. Initialise counters for each category to zero.
  2. Get today's date.
  3. Traverse the linked list in a single pass.
  4. For each node, increment the matching category counter and check date for today's count.
  5. Track the last node for "last event".
  6. Find the category with the maximum count.
  7. Display all statistics.
Pseudo Code
function showStatistics():
    categories[] ← {Phone:0, Travel:0, ...}
    todayCount ← 0
    today ← getCurrentDate()
    temp ← head
    last ← NULL

    while temp ≠ NULL:
        categories[temp.category]++
        if temp.date == today: todayCount++
        last ← temp
        temp ← temp.next

    maxCat ← max(categories)
    print "Total:", event_count
    print "Today:", todayCount
    print "Most Frequent:", maxCat
    print "First:", head.name
    print "Last:", last.name

Exit & Cleanup
Time: O(n)Space: O(1)▼
Algorithm
  1. Traverse the linked list from head.
  2. For each node, save next pointer, then free the current node.
  3. Traverse the stack from top.
  4. For each stack node, free the event data copy, then free the stack node.
  5. Set head = NULL, stack_top = NULL.
  6. Print goodbye message.
Pseudo Code
function cleanup():
    // Free linked list
    temp ← head
    while temp ≠ NULL:
        next ← temp.next
        free(temp)
        temp ← next
    head ← NULL

    // Free stack
    sTemp ← stack_top
    while sTemp ≠ NULL:
        sNext ← sTemp.next
        free(sTemp.event_data)
        free(sTemp)
        sTemp ← sNext
    stack_top ← NULL
