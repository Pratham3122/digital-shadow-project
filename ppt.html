Title

Digital Shadow

"Your Life Has a Memory, Even When You Don't."

A Digital Life Timeline Reconstruction System

2
Problem

Problem Statement

Humans generate thousands of digital events daily but forget the sequence and connections. Existing tools don't provide structured reconstruction or undo capabilities.

3
Objectives

Project Objectives

  • Build a timeline reconstruction system
  • Implement using Linked List & Stack
  • Enable undo and replay features
  • Provide statistical analysis
4
Architecture

System Architecture

User→ UI→ Linked List→ Stack→ Output
5
Data Structure

Linked List

Dynamic storage for events. Each node holds event data and a pointer to the next node. Supports O(n) insertion, deletion, and search.

6
Data Structure

Stack

LIFO structure for undo and replay operations. Push records actions, Pop reverses them. Implemented via a linked list for dynamic sizing.

7
Flow

Program Flowchart

Start → Menu → Choice → Execute Operation → Loop. 10 operations including add, delete, edit, search, display, replay, undo, and statistics.

8
Algorithms

Key Algorithms

  • Add Event — O(n) tail insertion
  • Delete — O(n) search + unlink
  • Replay — O(n) push all + pop all
  • Undo — O(1) pop + O(n) delete
9
Implementation

Implementation

Written in C with struct-based nodes, malloc/free memory management, modular functions, and cross-platform console UI. 530+ lines, zero warnings.

10
Results

Demo & Results

Successfully tested with 15+ events across 8 categories. All operations functional: add, delete, edit, search, display, replay, undo, statistics.
